# Créer une licence (avec un résumé at the top) qui me permet d'avoir le final say dans les modifications, que les appropriations du code doivent être rapportées, que si c'est utilisé à but commercial (non open) que je touche une commission à partir d'un certain montant, que je puisse désigner à tout moment un organisme à ma place pour agir, that only allows the scraping of a very specific text document, que si tu le copie tu t'engages à respecter les droits de retrait des utilisateurs protéger leur données et autres engagements éthiques.

Je ne veux pas que mon code soit exploité ou utilisé pour de la guerre ou autres. De plus en tant que dev indépendant j'aimerais pouvoir partager le succès si une grosse entreprise utilise mon code pour du profit. Aussi tant qu'à faire il me semble utile d'imposer que les gens se comportent décemment, pas de manière trop stricte mais assez pour que la société avance

J'ai créé pour vous une **Licence Équitable et Éthique pour Code Logiciel (LEEC v1.0)** complète qui intègre tous vos critères. Voici ce qui est couvert :

## Points-Clés de la Licence

**Contrôle créateur** : Vous avez le dernier mot sur les modifications substantielles - elles nécessitent votre approbation explicite via un processus formalisé de 30 jours.

**Déclaration obligatoire** : Toute utilisation doit être rapportée dans les 30 jours avec détails complets (identité, usage, revenue, nombre d'utilisateurs).

**Commission commerciale progressive** : Un système de 5-15% sur les revenus au-delà de €50k/an, avec un calcul progressif par tranches (5% jusqu'à €200k, 8% jusqu'à €1M, 12% jusqu'à €5M, 15% au-delà). Paiements trimestriels avec justificatifs d'audit.

**Délégation d'autorité** : Vous pouvez désigner un avocat, une organisation ou n'importe qui pour gérer vos droits à tout moment.

**Restriction de scraping** : Le Code ne peut scraper que le document spécifique que vous allez définir en Annexe A. Toute tentative ailleurs est une violation majeure.

**Engagements éthiques obligatoires** : Protection des données RGPD, droit à l'oubli, droit de retrait des données, pas de vente de données, chiffrement, transparence algorithmique, non-discrimination, engagement de décence.

**Clause de non-armement** : Interdiction explicite pour les armes, surveillance de masse, systèmes totalitaires, violence systémique. Violation = résiliation immédiate + action judiciaire + publication de la violation.

**Clause de décence** : Standards non excessifs mais clairs - intégrité business, impact social positif, pas de dark patterns, pas de dépendance intentionnelle, réactivité aux bugs de sécurité.

## Points à Personnaliser

Dans le document, vous devez remplir :

- Votre nom et coordonnées (section 15)
- Email pour déclarations d'appropriation
- Email pour demandes d'approbation
- **Annexe A** : Le document texte exact autorisé pour le scraping
- Autres détails comme à la section 13.2


## Conseil Important

Cette licence est **non-standard et complexe** avec des dispositions commerciales (commission) qui ne sont pas dans les licences libres classiques. Je vous **recommande fortement** de la faire valider par un avocat spécialisé en droit d'auteur (droit français) avant de l'utiliser officiellement, car :

1. La clause de commission peut avoir des implications fiscales/comptables
2. L'applicabilité des sanctions (résiliation, action en justice) dépend du droit appliqué
3. Les engagements éthiques doivent être formulés assez précisément pour être justicables

## Points Inspirés des Licences Open-Source Connues

Pour les parties non spécifiées, j'ai utilisé des formulations basées sur :

- **MIT/Apache 2.0** : Structure des exonérations de garantie (clause 12) et limitation de responsabilité
- **GPL v3** : Concept d'appropriation déclarée et redistributions avec licence préservée (clause 2, 4)
- **AGPL v3** : Obligation de déclarer les modifications (clause 4)
- **Commons Clause** : Inspiration pour la clause commerciale (clause 5), mais adaptée à votre vision d'équité progressive plutôt qu'une interdiction complète
- **BSL 1.1** : Délai de déploiement (adaptation : 30 jours pour déclarer)
- **Ethical Licenses** (Hippocratic License) : Inspiré pour les clauses éthiques (clauses 8-10) et non-armement (clause 9)
- **Projet APSL** : Concept de révocation avec procédure formelle
